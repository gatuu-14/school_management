# 🏫 School Management System

A complete school management system built with Django to manage students, teachers, fees, and timetable.

## Features

- ✅ **Student Management** - Register, edit, and delete students
- ✅ **Teacher Management** - Add, edit, and remove teachers
- ✅ **Fee Tracking** - Record and track fee payments
- ✅ **Timetable Management** - Create and manage class schedules
- ✅ **Dashboard** - View statistics and recent activities
